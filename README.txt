This plugin lets prospective buyers subscribe to a product when it is out of stock
== Description ==

[Live Demo](http://price-alert.dislbd.xyz/product/wordpress-pennant/) 

*Custom Email Sender / Subject / Email address
*Custom Email Message with tags (%product_name%, %product_link%)
*Custom Placeholder & Submit button value for the form
*Custom Error / Success message
*Disable default CSS
*Use shortcode instead of the default form
*List of subscribers with filters
